# Exam Management System

## 📌 Description
A full stack web application to manage exams, student results, attendance, and report cards.

## 🚀 Features
- Create Exams
- Add Results (Pass/Fail + Grade)
- Attendance Management
- Report Card Generation
- PDF Download
- Dashboard (Students, Exams, Pass %)
- Search, Sorting & CSV Export

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript, Tailwind CSS  
- Backend: Node.js, Express.js  
- Database: MongoDB Atlas  

## ▶️ Run Project

### Backend
cd server  
npm install  
node server.js  

### Frontend
Open index.html using Live Server  

## 🌐 Deployment
- Frontend: Netlify  
- Backend: Render
